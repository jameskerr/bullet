# Publish Script

After you've made your changes, go through these steps to publish the package.

## Run the Tests

```
yarn test
```

You can also watch for changes and run the tests with this command.

```
yarn test-watch
```

## Update the Types

The types are hand-coded in the d.ts file.

```
edit: bullet.d.ts
```

## Bump the Version

```
edit: pacakge.json
```

## Commit & Push to Main

## Publish to npm

```
npm publish
```
